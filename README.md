# Agent Skills

[![skills.sh](https://skills.sh/b/ahmad2b/agent-skills)](https://skills.sh/ahmad2b/agent-skills)

Reusable skills for AI agents.

## Skills

- `visual-creator` — create polished editorial visuals, article featured images, blog covers, and social graphics for publishing.

## Install

List available skills:

```bash
npx skills add ahmad2b/agent-skills --list
```

Install `visual-creator`:

```bash
npx skills add ahmad2b/agent-skills --skill visual-creator
```

Install all skills:

```bash
npx skills add ahmad2b/agent-skills --all
```
